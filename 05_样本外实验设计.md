# 样本外实验设计

## 1. 为什么你的样本外可能不是真样本外

很多研究者说自己做了样本外，但实际上测试集已经被污染。

常见过程：

```text
第一次测试集效果不好
-> 回去改因子
-> 再看测试集
-> 再改参数
-> 最后测试集变好
```

这时测试集已经变成训练集。

真正的样本外要求：

```text
在看到结果之前，研究规则已经固定。
```

## 2. 样本外污染的形式

### 直接污染

- 用测试集调参。
- 用测试集选因子。
- 用测试集决定持有期。
- 用测试集决定股票池过滤。

### 间接污染

- 读过测试期市场表现后设计因子。
- 看过测试集失败后修改异常值处理。
- 用全样本统计量标准化。
- 用全样本相关性筛选特征。
- 用全样本决定模型结构。

间接污染更隐蔽，也更常见。

## 3. Walk-forward 设计

更接近实盘的设计：

```text
用过去窗口训练/选择参数
只预测下一个未来窗口
窗口向前滚动
```

示例：

```text
2014-2018 训练 -> 2019Q1 测试
2014Q2-2019Q1 训练 -> 2019Q2 测试
...
```

关键原则：

- 每个测试点只能用过去数据。
- 特征筛选也必须在训练窗口内完成。
- 参数选择不能偷看未来窗口。

## 4. Purged CV 与 Embargo 的直觉

金融标签经常重叠。如果训练样本和测试样本的收益窗口重叠，即使日期不同，也可能泄露。

Purged CV 的思想：

```text
删除和测试标签窗口重叠的训练样本。
```

Embargo 的思想：

```text
测试集之后留出一段隔离区，不让紧邻样本进入训练。
```

这两个方法不是为了复杂，而是为了避免标签窗口泄露。

## 5. 市场状态样本外

时间样本外还不够，还要看市场状态。

至少分：

- 牛市。
- 熊市。
- 震荡市。
- 高波动。
- 低波动。
- 流动性充裕。
- 流动性紧张。
- 大盘风格。
- 小盘风格。

一个策略如果只在小盘牛市有效，就不能被描述为通用 alpha。

## 6. 股票池样本外

同一个因子应测试：

- 全 A。
- 沪深 300。
- 中证 500。
- 中证 1000。
- 高流动性股票池。
- 剔除最低流动性股票池。
- 不同板块。

目的不是要求所有股票池都强，而是理解它的适用边界。

## 7. 参数稳定性地图

不要只报告最优参数。应该画出参数区域：

```text
lookback × holding_period
```

观察：

- 是否存在一片稳定有效区域。
- 最优点是否孤立。
- 参数变化后收益是否平滑。
- 是否和因子逻辑一致。

稳定区域比最高点更重要。

## 8. 样本外判决

### 通过

- 固定规则后样本外仍有同方向效果。
- 收益下降但没有反向。
- 参数附近区域仍有效。
- 不同股票池表现符合逻辑。
- 市场状态表现可解释。

### 降级

- 样本外变弱明显，但仍有残余。
- 只在部分股票池或状态下有效。
- 需要重新定义适用范围。

### 失败

- 样本外反向。
- 参数稍变即失效。
- 只在已知历史阶段有效。
- 测试集被反复使用。

## 10. 训练任务

选一个旧项目，假设今天要重新做样本外：

1. 写下你已经看过哪些历史结果。
2. 承认哪些数据已经被污染。
3. 设计一个未来 3-6 个月模拟实盘验证。
4. 在验证开始前冻结规则。
5. 写清楚未来什么结果算通过，什么结果算失败。
