# 因子尸检案例库

## 1. 什么是因子尸检

因子尸检不是因子检验。因子检验问：

```text
这个因子是否有效？
```

因子尸检问：

```text
这个因子为什么看起来有效？它真正赚的是什么？
```

很多因子并不是完全没用，而是被误解了。它可能不是 alpha，而是：

- 小市值风险溢价。
- 低流动性补偿。
- 行业轮动。
- 困境股反转。
- 数据泄露。
- 成本未扣除的微弱信号。
- 特定市场阶段的风格暴露。

尸检的目的不是羞辱策略，而是给它准确命名。

## 2. 案例一：小市值伪 alpha

### 表面现象

某因子分组收益很好，高分组长期跑赢低分组。

但进一步检查发现：

```text
高分组平均市值显著低于低分组
多头组合集中在小市值股票
市值中性后 IC 接近 0
```

### 可能真相

因子本身没有独立信息，它只是选到了小票。

这不一定完全无价值，因为小市值可能在某些时期有风险溢价。但如果你把它说成 alpha，就错了。

### 必做尸检

- 分市值组计算 IC。
- 控制 log 市值后再计算 IC。
- 多头组合与基准的市值暴露。
- 小市值指数收益与策略收益相关性。
- 小票极端行情中策略表现。

### 死亡标准

```text
市值中性后 IC 和分组收益基本消失。
```

### 保留标准

```text
大中小市值组内均有一定 IC，或市值中性后仍有残余收益。
```

## 3. 案例二：低流动性陷阱

### 表面现象

低流动性股票未来收益更高，分组收益明显。

### 可能真相

这是流动性风险补偿，不是可免费获取的 alpha。低流动性股票的纸面收益可能无法以合理规模成交。

### 必做尸检

- 分成交额组 IC。
- 剔除最低 20% 流动性股票后重测。
- 成本 2 倍、3 倍压力测试。
- 按资金规模估算 participation rate。
- 检查涨跌停和停牌贡献。

### 死亡标准

```text
收益主要来自最低流动性组，且成本/容量测试后消失。
```

### 保留标准

```text
在中高流动性股票中仍有收益，且容量估算可接受。
```

## 4. 案例三：行业暴露伪 alpha

### 表面现象

因子整体 IC 为正，分组收益好。

进一步发现：

```text
高分组长期集中在新能源、医药、TMT 等特定行业
行业中性后收益大幅下降
```

### 可能真相

因子捕捉的是某段时期的行业景气或行业轮动，而不是行业内选股能力。

### 必做尸检

- 行业内 RankIC。
- 行业中性前后对比。
- 分行业 IC。
- 多头组合行业权重时间序列。
- 行业收益归因。

### 死亡标准

```text
行业内完全无效，收益主要来自行业配置。
```

### 保留标准

```text
行业配置贡献存在，但行业内仍有稳定选股能力。
```

## 5. 案例四：公告日泄露型基本面因子

### 表面现象

盈利增长、ROE 改善、分析师上调等因子表现非常强。

### 可能真相

财报或分析师数据提前使用了。特别是当因子在公告前就表现强烈时，要高度怀疑。

### 必做尸检

- 按公告日滞后 1、5、20 日重测。
- 排除公告日前后短窗口收益。
- 检查数据供应商入库时间。
- 对比报告期口径和公告日口径。
- 检查是否使用了修订后数据。

### 死亡标准

```text
按公告日延迟后因子大幅坍塌。
```

### 保留标准

```text
严格公告日处理后仍有中长期效果。
```

## 6. 案例五：反转因子的交易不可实现收益

### 表面现象

短期反转因子 1-5 日收益显著。

### 可能真相

收益来自开盘跳空、涨跌停后修复、低价小票噪声，实际成交很难获取。

### 必做尸检

- t+1 open、VWAP、close 不同成交价对比。
- 延迟一天交易。
- 剔除涨跌停股票。
- 剔除低价股。
- 剔除最低成交额组。
- 成本敏感性。

### 死亡标准

```text
延迟一天或加入真实成本后收益消失。
```

### 保留标准

```text
在可成交假设下仍有稳定收益。
```

## 7. 案例六：机器学习模型学到了缺失模式

### 表面现象

模型加入大量特征后效果明显提升。

### 可能真相

模型不是学到了 alpha，而是学到了：

- 某些数据只有大公司有。
- 某些字段缺失代表公司质量差。
- 数据覆盖范围随时间变化。
- 数据供应商回填造成时间泄露。

### 必做尸检

- 缺失率特征重要性。
- 只用缺失哑变量训练模型。
- 按数据覆盖组分层测试。
- 按时间检查覆盖率变化。
- 删除高泄露风险字段后重测。

### 死亡标准

```text
模型主要收益来自缺失模式或覆盖偏差。
```

### 保留标准

```text
删除缺失 proxy 后仍有稳定边际提升。
```

## 8. 因子尸检报告模板

```text
因子名称：
表面表现：
最可能伪来源：
尸检实验：
最致命发现：
是否仍有残余 alpha：
应改名为：
最终判决：
```

“应改名为”很重要。例如：

```text
原名：质量成长因子
尸检后：小市值成长暴露
```

命名准确，研究才不会自欺。
