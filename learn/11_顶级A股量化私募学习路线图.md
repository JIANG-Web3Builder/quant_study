# 顶级 A 股量化私募学习路线图

## 1. 先把目标说清楚

你想要的不是“再多学几个因子”，而是逐步具备下面这类能力：

```text
能提出研究假设
-> 能用 point-in-time 数据严谨验证
-> 能识别伪 alpha 和风险暴露
-> 能把信号转成真实可交易组合
-> 能在 A 股制度约束下估算成本和容量
-> 能持续监控衰减并下线失效策略
```

顶级量化私募真正稀缺的，不只是某个神秘信号，而是：

- 数据纪律。
- 研究设计。
- 统计判断。
- 组合工程。
- 执行约束认知。
- 实盘监控与研究管理。

所以这份路线图不会把重点放在“收集因子”，而会把重点放在：

```text
建立能反复产出、反复证伪、反复迭代的研究系统。
```

## 2. 你现在大概处在什么位置

结合你现有文档和你的自我描述，你现在更接近：

- 会做基础研究和回测。
- 对因子、择时、组合、回测已经有基本概念。
- 已经开始接触更严肃的研究问题。
- 不是纯入门，但还没有形成机构级的完整能力闭环。

你当前最明显的优势是：

- 已经开始重视 `数据泄露`。
- 已经开始重视 `过拟合` 和 `样本外污染`。
- 已经开始重视 `成本`、`容量`、`组合工程`。
- 已经有了红队研究意识。

你当前最关键的短板通常会在这里：

- 知识是碎片化的，不是能力树化的。
- 会做实验，但不一定有完整研究工序。
- 会看单个因子，但不一定能把它稳定转成组合。
- 会看回测，但不一定能设计模拟实盘和衰减监控。
- 可能学过很多术语，但还没把它们压成自己的研究动作。

所以正确路径不是“补更多概念”，而是：

```text
把零散知识变成稳定的研究生产线。
```

## 3. 顶级量化私募能力树

### 3.1 第一层：市场制度与数据层

这是最底层。如果这一层不稳，后面所有显著性都不可信。

必须会：

- A 股涨跌停制度。
- 停牌、复牌、ST、退市处理。
- 公告日、报告期、入库时间的区别。
- 指数历史成分股与幸存者偏差。
- 行业分类的历史版本问题。
- 前复权、后复权、真实交易价格的区别。
- 收盘后信号与成交时点的关系。
- 一字板、跌停、停牌对成交可实现性的影响。

进阶要会：

- 数据修订与供应商回填风险。
- 研究字段级 point-in-time 审计。
- 不同数据源之间的时间戳差异。
- 另类数据和分析师数据的可见时间控制。

没学到位的信号：

- 你说不清某个字段在历史上何时可见。
- 你默认财报按报告期可用。
- 你默认当前成分股可以回测过去。
- 你用复权价格处理交易约束。

### 3.2 第二层：统计与研究设计层

这一层决定你能不能区分“看起来有效”和“真的有证据”。

必须会：

- 假设检验的直觉，而不是只会背 p-value。
- 多重检验问题。
- 标签重叠与自相关。
- 横截面相关与有效样本数缩水。
- Walk-forward。
- Purged CV / Embargo 的直觉和适用场景。
- 参数稳定性图，而不是只看最优点。
- 时间块分析、分年份分析、随机化测试。
- Newey-West 一类保守统计口径的思想。

进阶要会：

- Bootstrap 在金融样本中的使用边界。
- Deflated Sharpe 的直觉。
- regime split 和 state conditional evaluation。
- 研究过程打折，而不是只给结果打分。

没学到位的信号：

- 只报样本内最优结果。
- 不记录失败实验。
- 测试集被反复用来改策略。
- 看到高夏普就默认策略强。

### 3.3 第三层：Alpha 研究层

这里才是很多人以为的“量化核心”，但实际上它建立在前两层之上。

必须会：

- 如何从经济机制或行为机制提出假设。
- 如何把假设变成可计算特征。
- 如何判断一个因子到底赚什么。
- 如何识别它只是市值、行业、流动性、波动率、动量等暴露。
- 如何把基本面、价格量、事件类信号放到同一研究框架下。
- 如何判断信号半衰期。
- 如何定义它适用的股票池、持有期和市场状态。

进阶要会：

- 多因子组合而不是因子堆砌。
- 状态依赖 alpha。
- 事件驱动和横截面因子的统一表达。
- hypothesis factory：持续产出研究假设的能力。

没学到位的信号：

- 因子名字很好听，但机制说不清。
- 中性化后完全死亡，却仍被称为 alpha。
- 只会找公开因子，不会自己生成研究问题。

### 3.4 第四层：组合与风险层

很多策略死在这里：单因子有效，组合后无效。

必须会：

- score 不等于预期收益。
- 组合构建会扭曲信号。
- 风险模型会漏项、会有估计误差。
- 优化器会放大噪声。
- 约束会吃掉 alpha。
- 组合收益可能来自少数股票。
- 预测风险和实现风险必须对照。

进阶要会：

- 风格、行业、市值、流动性暴露控制。
- 协方差矩阵 shrinkage 思想。
- 风险预算思维。
- 多信号、多策略相关性管理。
- 组合层 alpha 保真分析。

没学到位的信号：

- 优化后收益突然变得极漂亮，但解释不清。
- 组合高度集中且依赖少数股票。
- 约束一加就失效，但不知道是策略问题还是组合问题。

### 3.5 第五层：执行、成本与容量层

这层决定你的东西到底是纸面收益还是可交易收益。

必须会：

- 成本不是常数。
- 参与率与冲击成本的直觉。
- 延迟成交测试。
- A 股涨跌停和停牌约束。
- 资金规模阶梯下的容量评估。
- 低流动性收益和真实 alpha 的区别。
- 换手、alpha 半衰期、成本三者的关系。

进阶要会：

- 拥挤交易风险识别。
- 成本曲线而不是单点成本。
- 模拟未成交与目标偏离。
- 信号衰减和执行延迟联动分析。

没学到位的信号：

- 只测一个 10bp 成本。
- 不看参与率。
- 不测延迟成交。
- 收益主要来自最低流动性股票。

### 3.6 第六层：工程与研究流程层

顶级机构和普通学习者最大的差距，经常不在公式，而在流程。

必须会：

- 研究日志。
- 失败实验保留。
- 字段级数据审计。
- 可复现回测。
- 统一实验命名。
- 研究报告模板化。
- 模拟实盘监控。
- 策略下线条件预先定义。

进阶要会：

- point-in-time 数据层建设。
- 特征、标签、交易假设版本管理。
- 模型/因子衰减监控。
- 研究优先级和资源配置。
- 从单个研究项目走向研究管线管理。

没学到位的信号：

- 同一策略改了很多版，但自己说不清差异。
- 失败实验没有记录。
- 每次回测都像重新来过。
- 没有未来什么情况算失效的规则。

### 3.7 第七层：研究员成熟度层

顶级研究员和普通研究者的最终差别，是判断力。

必须形成的习惯：

- 先怀疑，再证明。
- 先问机制，再问结果。
- 先问边界，再问平均表现。
- 先问成本和容量，再问纸面收益。
- 先想失效方式，再想上线。

进阶目标：

- 能快速判断一个想法值得不值得投入。
- 能从一堆失败研究里提炼规律。
- 能解释为什么这个策略应该存在、何时失效、何时下线。
- 能用机构语言写报告，而不是写“我觉得这个因子还可以”。

## 4. 6-12 个月最短闭环主线

如果你时间有限，这条线最重要。

### 阶段 1：第 1-2 个月

目标：把研究底座打稳，把 A 股制度和 point-in-time 纪律变成习惯。

核心知识：

- 公告日、报告期、入库时间。
- 成分股历史、退市、停牌、涨跌停处理。
- 信号时点和成交时点的严格对应。
- 样本外、验证集、测试集的边界。

必须做的项目：

1. 给你当前常用的 20 个核心字段写一份 `字段可见时间审计表`。
2. 选一个旧基本面项目，按公告日滞后 1、5、20 日重测。
3. 选一个旧价格量项目，做 `t+1 open`、`t+1 VWAP`、`t+2 open` 成交对照。
4. 用现有文档给一个你最满意的旧策略做完整红队审计。

阶段验收：

- 你能说清每个核心字段何时可见。
- 你能承认哪些旧结果已经被污染。
- 你至少杀死 1 个自己曾经喜欢的旧策略。

常见伪进步：

- 读了很多 PIT 概念，但没做字段审计表。
- 口头承认不能看未来，但回测里仍默认可成交。
- 继续把污染测试集叫测试集。

建议配合文档：

- `00_项目使用方式.md`
- `01_如何红队审计一个量化策略.md`
- `03_数据泄露取证.md`
- `05_样本外实验设计.md`

### 阶段 2：第 3-4 个月

目标：形成机构化的单因子研究框架，而不是随便试因子。

核心知识：

- 研究假设写法。
- 基本面因子、价格量因子、事件因子的差别。
- 因子尸检。
- 风格暴露和伪 alpha。
- 参数稳定性与分组单调性。

必须做的项目：

1. 重做 1 个基本面因子，要求：
   - 严格按公告日口径。
   - 做行业中性、市值中性前后对比。
   - 做分市值组、分行业组测试。
2. 重做 1 个价格量因子，要求：
   - 成本曲线。
   - 延迟成交。
   - 剔除最低流动性组后重测。
3. 写 1 份因子尸检报告，把一个“好听的因子名”改成“真实来源名”。

阶段验收：

- 你至少有 2 个经过尸检后仍有残余效果的信号。
- 你能解释它们更像什么，不像什么。
- 你能说明它们适用于什么股票池、持有期和市场状态。

常见伪进步：

- 因子看起来多，但都没做中性化和尸检。
- 只会展示最好的分组图。
- 讲故事时比做验证时更自信。

建议配合文档：

- `02_回测过拟合与虚假显著性.md`
- `04_因子尸检案例库.md`
- `05_样本外实验设计.md`

### 阶段 3：第 5-6 个月

目标：从“因子有效”升级到“策略可进入组合测试”。

核心知识：

- Walk-forward。
- 参数稳定区域。
- 组合构建方式对 alpha 的扭曲。
- 约束、风险暴露、换手和成本的联动。

必须做的项目：

1. 对阶段 2 留下的 2 个信号做完整 walk-forward。
2. 对每个信号做：
   - 参数扰动图。
   - 成本 1 倍、2 倍、3 倍压力测试。
   - 股票池外推测试。
   - 市场状态分层测试。
3. 用至少两种组合方法对比：
   - 简单分层组合。
   - 有约束组合构建。
4. 写 1 份机构级 Alpha 报告。

阶段验收：

- 至少 1 个信号在更保守假设下仍值得 `继续研究` 或 `进入组合测试`。
- 你能分清：
  - 因子问题
  - 组合问题
  - 成本问题
  - 容量问题

常见伪进步：

- 因子很好，但组合一做就坏掉。
- 组合漂亮，但靠的是风险模型漏洞或极端持仓。
- 样本外表现差，却继续靠调参挽救。

建议配合文档：

- `05_样本外实验设计.md`
- `06_组合工程病理学.md`
- `07_执行成本与容量压力测试.md`
- `09_机构级Alpha研究报告模板.md`

### 阶段 4：第 7-9 个月

目标：建立多信号组合和状态依赖研究，不再停留在单因子世界。

核心知识：

- 多因子组合的相关性管理。
- 行业、风格、流动性暴露的统一看法。
- 市场状态切换下的因子表现。
- 信号半衰期差异和调仓节奏。

必须做的项目：

1. 把前面留下的信号组合成一个多信号框架。
2. 增加一个状态层：
   - 例如高波动 / 低波动。
   - 大盘风格 / 小盘风格。
   - 风险偏好强 / 风险偏好弱。
3. 比较：
   - 静态权重组合。
   - 状态条件权重组合。
4. 做信号相关性、收益归因、风险暴露分解。

阶段验收：

- 组合不是简单叠加，而是知道每个信号贡献什么。
- 你能解释为什么某些信号只应在特定状态下提高权重。
- 你能说清组合收益里哪些是 alpha，哪些是风格暴露。

常见伪进步：

- 多因子组合只是“把能赚钱的都加进去”。
- 组合后看起来更稳，但其实是暴露更多了。
- 状态划分只是事后解释，不是事前规则。

### 阶段 5：第 10-12 个月

目标：进入模拟实盘研究范式，建立周度研究节奏。

核心知识：

- 模拟实盘和历史回测的差别。
- 监控指标设计。
- 下线规则和衰减跟踪。
- 研究优先级管理。

必须做的项目：

1. 选 1 个经过前四阶段仍存活的策略，做至少 3 个月模拟实盘跟踪。
2. 每周记录：
   - 滚动 RankIC。
   - 分组收益。
   - 风格暴露。
   - 换手和成本。
   - 目标持仓与实际持仓偏离。
3. 预先写好：
   - 继续条件。
   - 降级条件。
   - 下线条件。
4. 每周复盘 1 个旧项目，累计至少 12 个项目判决。

阶段验收：

- 你至少有 1 个能被严谨追踪的模拟实盘对象。
- 你至少完成 12 个旧项目红队复盘。
- 你开始拥有自己的“常见死亡原因清单”。

常见伪进步：

- 把模拟实盘当展示，不当监控。
- 没有预定义失效条件。
- 看到短期回撤就随意改规则。

## 5. 如果你只能抓主线，先学什么

优先级应该是：

```text
数据纪律
-> 研究设计
-> 因子尸检
-> 样本外与参数稳定性
-> 组合工程
-> 成本与容量
-> 模拟实盘监控
-> 多因子与状态模型
-> 机器学习
-> 高频与微观结构
```

这条顺序很重要。

很多人失败不是因为不够聪明，而是顺序错了：

- 基础数据纪律没稳就上 ML。
- 还不会做组合和容量，就开始追逐高频。
- 还没建立失败记录，就开始找更多新因子。

## 6. 2-3 年远期路线

### 第 1 年：把研究闭环做出来

目标：

- 建立严谨的 A 股 research loop。
- 能从想法走到报告。
- 能从回测走到模拟实盘。
- 能分清哪些项目该死，哪些项目该继续。

年底应该达到：

- 至少 12 个旧项目复盘记录。
- 至少 3 个完整因子/策略机构级报告。
- 至少 1 个模拟实盘跟踪对象。
- 一份自己的“研究红旗清单”和“常见死亡原因库”。

### 第 2 年：把组合、风险和平台化补齐

目标：

- 开始从单因子研究员走向组合研究员。
- 提升风险模型和组合优化理解。
- 搭更稳定的研究数据层和实验管理。
- 开始考虑多策略相关性和组合配置。

重点能力：

- 风格、行业、流动性、波动暴露管理。
- 组合层收益归因。
- 风险预算思维。
- 更高质量的回测和实验版本控制。
- 研究流程自动化。

年底应该达到：

- 至少 1 个多因子或多信号组合框架。
- 至少 1 套标准化研究管线。
- 至少 1 套模拟实盘监控看板指标体系。

### 第 3 年：向研究负责人或 PM 思维过渡

目标：

- 学会做研究优先级管理。
- 学会评估一个方向的预期投入产出比。
- 学会管理多个策略、多个状态、多个资金规模下的决策。
- 开始形成自己的方法论，而不是只借别人的框架。

重点能力：

- hypothesis generation 体系化。
- 研究资源配置。
- 策略组合层配置。
- 衰减监控和下线纪律。
- 对拥挤、容量、相关性做更高层的判断。

到这个阶段，你追求的不再是“找到一个好因子”，而是：

```text
让研究系统持续产出可审计、可交易、可监控的边际 alpha。
```

## 7. 阶段实践任务矩阵

### 任务 1：字段可见时间审计

训练能力：

- 数据纪律。
- point-in-time 思维。

产出：

- 一份字段级审计表。

通过标准：

- 每个核心字段都有可见时间、修订风险、使用时间说明。

### 任务 2：旧基本面项目重做

训练能力：

- 公告日口径处理。
- 延迟测试。

产出：

- 一份重测对照报告。

通过标准：

- 能明确说出原项目哪些结果是高估的。

### 任务 3：单因子尸检报告

训练能力：

## 8. 资源优先级清单

### A 级：必须优先啃

这些不是“看完就会赚钱”，但它们决定你能不能走正路。

- 你当前项目里的 `00-10` 文档。
- 上交所、深交所交易规则和相关制度说明。
- 指数编制与历史成分股规则材料。
- 财报披露、公告时点、停复牌、涨跌停等一手制度材料。
- `Active Portfolio Management`。
- `Expected Returns`。
- 与 purged CV、embargo、多重检验、研究过拟合相关的经典材料。

A 级使用原则：

```text
不是背诵，而是每学一点就反打到旧项目上。
```

### B 级：第 2 阶段后补

这些很重要，但不应抢在主线前面。

- 风险模型与组合优化更深入材料。
- `Advances in Financial Machine Learning` 的选读章节。
- `Machine Learning for Asset Managers`。
- 时间序列、横截面资产定价、稳健统计相关材料。

B 级使用原则：

```text
只在你已经有严格研究流程后再吸收，不要拿它们替代研究纪律。
```

### C 级：远期分支

这些值得学，但不是你当前最短闭环主线。

- 高频与微观结构。
- 执行算法细节。
- 更复杂的机器学习结构。
- 另类数据的大规模工程化。

C 级使用原则：

```text
只有当你已经能稳定完成中低频 A 股研究闭环后，再把它们当加速器，而不是救命稻草。
```

## 10. 学习过程中最危险的幻觉

### 幻觉一：顶级私募的优势主要是隐藏因子

更接近事实的是：

- 他们更强的是研究流程、数据质量、执行纪律、风险控制和组织化能力。

### 幻觉二：我只要学更多数学就会变强

数学重要，但在你当前阶段，更大的提升通常来自：

- 数据纪律。
- 样本外设计。
- 研究记录。
- 组合和成本认知。

### 幻觉三：机器学习比传统因子更高级

现实经常是：

- 复杂模型更容易把泄露、缺失模式、风格暴露学进去。

### 幻觉四：回测好就值得继续做

更好的问题是：

- 它在红队后还剩下什么？

### 幻觉五：我还没准备好做模拟实盘

如果你永远不进入追踪阶段，你就会一直停在“漂亮历史结果”的舒适区。

## 11. 什么时候算你真的进阶了

### 12 个月后，至少应该做到

- 能用机构化方式写 3 份完整研究报告。
- 能对一个策略做完整红队审计。
- 能独立做字段可见时间审计。
- 能把因子和组合、成本、容量联系起来看。
- 能解释 1 个模拟实盘对象的监控逻辑。
- 能积累至少 12 个旧项目判决。

### 2-3 年后，理想上应该做到

- 能同时管理多个研究方向，而不是只盯一个因子。
- 能搭出更稳定的研究和实验流程。
- 能把多信号转成更稳健的组合框架。
- 能对策略相关性、容量、拥挤和衰减做更成熟判断。
- 能在没有外界喂题的情况下，持续生成可验证研究假设。

## 12. 最后的路线原则

请把下面这句话当成这条学习路线的总纲：

```text
不要追求最快找到一个看起来赚钱的策略。
要追求尽快建立一套能持续杀死假策略、保留少量真东西的研究系统。
```

普通学习者成长慢，经常是因为：

- 不愿意承认旧项目死亡。
- 不愿意保留失败记录。
- 不愿意面对成本、容量和样本外衰减。
- 不愿意让研究过程比结果更重要。

而真正接近顶级研究员的路径，恰恰相反：

```text
更严厉地审计自己
-> 更早地承认错误
-> 更慢地放行策略
-> 更长期地积累判断力
```

如果你能连续一年按这条路线执行，你得到的最大收获通常不是“突然拥有一个圣杯策略”，而是：

- 更强的研究判断力。
- 更干净的数据和实验习惯。
- 更少自欺。
- 更高质量的策略存活率。
- 更接近机构级量化研究的真实工作方式。
